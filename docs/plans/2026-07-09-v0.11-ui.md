## Global constraints

  `<script>try{var t=localStorage.getItem("rp-board:theme");var d=t?t==="dark":matchMedia("(prefers-color-scheme: dark)").matches;if(d)document.documentElement.classList.add("dark")}catch(e){}</script>`
- `board/src/lib/theme.ts`: `export type Theme = "light" | "dark"; export function resolveTheme(stored: string | null, systemDark: boolean): Theme` (stored value wins when "light"/"dark", else system). `export const THEME_KEY = "rp-board:theme"`.

### Task 3: Soft-unwrap
- `board/src/lib/markdownText.ts`: `export function unwrapSoftBreaks(md: string): string`. Split lines; track fenced code (```/~~~ toggles) — never join inside; also skip when current line is indented-code (4+ spaces) while previous blank. Join rule: join `lines[i+1]` onto `lines[i]` with a single space when: `lines[i]` non-empty, does not end with two spaces or backslash, is not a table row (`^\s*\|`), heading, blockquote, or fence line; AND `lines[i+1]` matches `^[a-z(]` and is not an ordered-list line (`^\s*\d+[.)]\s`). Iterate until stable within the joined line (a 3-line wrap collapses fully).
- `board/src/lib/markdownText.test.ts`: BK's exact example joins to one paragraph; `Serves:`/`Signed off:`/`RQ1:` lines keep breaks; list items keep breaks; wrapped list-item continuation joins (`- long text\ncontinued` → joined); fenced code untouched; two-space break kept; table rows untouched; uppercase next line kept.
- `Markdown.tsx`: `marked.parse(unwrapSoftBreaks(source))`.

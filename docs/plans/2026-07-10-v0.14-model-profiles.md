**Goal:** A per-project model profile (`plans/model-profile.md`) that nudges interactive stages toward the profile's model and pins delegated stages via generated project agents (`.claude/agents/rp-*.md`), per the v0.14 section of `docs/specs/2026-07-09-updates-sharing-models-design.md`.

Command markdown files gain minimal, surgical text: three nudge blocks (plan/sync/SKILL), three dispatch changes (results/board/review), one new command (`models.md`), and one init hook.

**Tech Stack:** Python 3 stdlib only (no pip), `unittest` (run via `python3 -m unittest`), markdown command/agent files. No board/TypeScript changes in this release ("No board changes" — spec).

## Global Constraints

- **Platform facts (verified 2026-07-10 against code.claude.com/docs/en/sub-agents.md, on Claude Code 2.1.206):**
  - Agent frontmatter supports `model:` (aliases `sonnet`/`opus`/`haiku`/`fable`, full `claude-*` IDs, `inherit`; omitted = inherit) and `effort:` (`low`|`medium`|`high`|`xhigh`|`max`).
  - Agent `tools:` is a comma-separated list of **bare tool names only** — permission specifiers like `Bash(git:*)` are NOT supported. The spec's least-privilege `Bash(git:*)` for rp-plan-reviewer is therefore expressed as `tools: Read, Grep, Glob, Bash` plus an explicit body constraint ("Bash is for read-only git evidence gathering only"). Record this as a spec deviation (Task 9).
  - Named project agents in `.claude/agents/` are dispatchable via the Agent (Task) tool's `subagent_type`; precedence is managed > project > user > plugin, so project `rp-*` files shadow any plugin agent of the same name.
  - Model resolution order: `CLAUDE_CODE_SUBAGENT_MODEL` > per-invocation `model` param > agent frontmatter > session model, filtered by the org `availableModels` allowlist. **Dispatching commands must NOT pass a model override in the Task call** — that would defeat the frontmatter pin.
- **Version-neutral release:** CHANGELOG entries go under `[Unreleased]`; `.claude-plugin/plugin.json`, `.claude-plugin/marketplace.json`, and `board/package.json` stay at 0.13.0; no tags. BK numbers releases at cut.
- **Exact strings (copy verbatim, defined once in models.py / templates):**
  - Ownership marker: `<!-- generated by research-plans /models · profile sha256:<64 hex> -->`
  - Mismatch hint: `model profile changed since agents were generated — run /research-plans:models`
  - Nudge line: `` Model profile: this stage is set to <model>; you're on <current>. Switch with /model <model> (safe mid-conversation — nothing is lost), or continue as-is. ``
- **Fallback invariants:** no profile file → total silence, zero behavior change; malformed row → stderr warning at the point the stage runs + fall back to the session model; missing `rp-*` agent → today's behavior (anonymous subagent for results/board; fully inline for review). Malformed profiles must never block a stage.
- **Committed artifacts:** `plans/model-profile.md` and `.claude/agents/rp-*.md` are shared project configuration — committed, never gitignored by the plugin.
- **Scripts:** python3 stdlib only, `#!/usr/bin/env python3`, module-level docstring, same conventions as `check_update.py`. Tests: `unittest`, one file per script under `tests/`, header comment naming the run command.
- **Prose:** never hard-wrap; one paragraph per line (user CLAUDE.md rule 7).
- **Commits:** conventional prefixes as in recent history (`feat(models): …`, `docs(models): …`, `test(models): …`); no Co-Authored-By lines.

# Model profile

<!-- research-plans:model-profile -->

How each research-plans stage picks a Claude model. Two mechanisms — **nudge**: Claude tells you the profile's model for this stage and suggests `/model`; you decide. **agent**: this delegated stage runs on the profile's model automatically (best-effort — an org model allowlist, `CLAUDE_CODE_SUBAGENT_MODEL`, or a per-invocation override can supersede the request). `inherit` in a model cell means "whatever your session is using."

| stage | model | effort | mechanism |
|---|---|---|---|
| plan (co-authoring) | opus | max | nudge |
| execute (analysis) | sonnet | — | nudge |
| sync | inherit | — | nudge |
| plan review (verdict + grade) | opus | medium | agent |
| results validation | opus | low | agent |
| board reviewer panel | opus | low | agent |

Why these defaults: planning is where quality compounds, so it gets the strongest model at max effort. Execution is interactive and iterative, so a fast cheap model stretches subscription quota. Review and validation are short judgment tasks where a smarter prior catches what longer thinking on a weaker model misses — hence opus at low effort. `effort` on nudge rows is advisory; on agent rows it is written into the generated agent file.

After hand-editing this table, run `/research-plans:models` to validate it and regenerate the agents in `.claude/agents/` — a silent typo in a row otherwise just stops that stage's nudge or pin.
```

#!/usr/bin/env python3
"""research-plans model profile — parse plans/model-profile.md and generate
project review agents in .claude/agents/. Stdlib only.

Subcommands:
    stage <key>   print one stage's row as JSON (empty output = no profile or
                  no usable row; parse warnings go to stderr)
    generate      write rp-*.md agent files from the profile (ownership-marked,
                  checksum-stamped; refuses same-named files without the marker)
    check         print a one-line hint when generated agents are stale
                  against the current profile
"""
import argparse
import hashlib
import json
import re
import sys
from pathlib import Path

STAGE_LABELS = {
    "plan": "plan",
    "execute": "execute",
    "sync": "sync",
    "plan review": "plan-review",
    "results validation": "results-validation",
    "board reviewer panel": "board-reviewer",
}
MODEL_ALIASES = {"inherit", "opus", "sonnet", "haiku", "fable"}
MODEL_ID_RE = re.compile(r"^claude-[a-z0-9.-]+$")
EFFORT_LEVELS = {"low", "medium", "high", "xhigh", "max"}
NO_EFFORT = {"", "-", "—", "–"}  # blank, hyphen, em dash, en dash
MECHANISMS = {"nudge", "agent"}
AGENT_STAGES = {
    "plan-review": "rp-plan-reviewer",
    "results-validation": "rp-results-validator",
    "board-reviewer": "rp-board-reviewer",
}
MARKER_RE = re.compile(
    r"<!-- generated by research-plans /models · profile sha256:([0-9a-f]{64}) -->"
)
MISMATCH_HINT = (
    "model profile changed since agents were generated — run /research-plans:models"
)
PROFILE_REL = Path("plans") / "model-profile.md"


def find_root(start=None):
    p = Path(start if start else Path.cwd()).resolve()
    for cand in (p, *p.parents):
        if (cand / "plans" / "master-plan.md").exists():
            return cand
    return p


def _norm(cell):
    return re.sub(r"\s+", " ", re.sub(r"\([^)]*\)", "", cell).strip().lower())


def _row_cells(line):
    s = line.strip()
    if not s.startswith("|"):
        return None
    return [c.strip() for c in s.strip("|").split("|")]


def _add_row(stages, warnings, cells, rownum):
    if len(cells) != 4:
        warnings.append(
            f"model-profile: skipping row {rownum} (expected 4 cells, got {len(cells)})"
        )
        return
    raw_stage, raw_model, raw_effort, raw_mech = cells
    key = STAGE_LABELS.get(_norm(raw_stage))
    if key is None:
        warnings.append(f"model-profile: skipping row {rownum} (unknown stage {raw_stage!r})")
        return
    if key in stages:
        warnings.append(f"model-profile: skipping row {rownum} (duplicate stage {raw_stage!r})")
        return
    model = raw_model.strip().lower()
    if model not in MODEL_ALIASES and not MODEL_ID_RE.match(model):
        warnings.append(f"model-profile: skipping row {rownum} (unknown model {raw_model!r})")
        return
    effort = raw_effort.strip().lower()
    if effort in NO_EFFORT:
        effort = None
    elif effort not in EFFORT_LEVELS:
        warnings.append(f"model-profile: skipping row {rownum} (unknown effort {raw_effort!r})")
        return
    mech = raw_mech.strip().lower()
    if mech not in MECHANISMS:
        warnings.append(
            f"model-profile: skipping row {rownum} (unknown mechanism {raw_mech!r})"
        )
        return
    stages[key] = {"stage": key, "model": model, "effort": effort, "mechanism": mech}


def parse_profile(text):
    """Return (stages, warnings). Strict: malformed rows are warned and skipped."""
    stages, warnings = {}, []
    lines = text.splitlines()
    i = 0
    while i < len(lines):
        cells = _row_cells(lines[i])
        if cells is not None and [_norm(c) for c in cells] == ["stage", "model", "effort", "mechanism"]:
            i += 1
            if i < len(lines):
                sep = _row_cells(lines[i])
                if sep is not None and set("".join(sep)) <= set("-: "):
                    i += 1
            rownum = 0
            while i < len(lines):
                cells = _row_cells(lines[i])
                if cells is None:
                    break
                rownum += 1
                _add_row(stages, warnings, cells, rownum)
                i += 1
            break
        i += 1
    if not stages and not warnings:
        warnings.append("model-profile: no stage/model/effort/mechanism table found")
    return stages, warnings


def load_profile(root):
    """Return (stages, warnings, exists)."""
    path = root / PROFILE_REL
    if not path.exists():
        return {}, [], False
    try:
        text = path.read_text(encoding="utf-8")
    except (OSError, UnicodeDecodeError):
        return {}, ["model-profile: unreadable (not UTF-8 or IO error)"], True
    stages, warnings = parse_profile(text)
    return stages, warnings, True


def cmd_stage(root, key):
    stages, warnings, exists = load_profile(root)
    if not exists:
        return 0
    for w in warnings:
        print(w, file=sys.stderr)
    row = stages.get(key)
    if row:
        print(json.dumps(row))
    return 0


def main(argv=None):
    ap = argparse.ArgumentParser(description="research-plans model profile tools")
    ap.add_argument("--root", default=None, help="project root (default: walk up to plans/master-plan.md)")
    sub = ap.add_subparsers(dest="cmd", required=True)
    p_stage = sub.add_parser("stage", help="print one stage's profile row as JSON")
    p_stage.add_argument("key", choices=sorted(set(STAGE_LABELS.values())))
    sub.add_parser("generate", help="write rp-*.md agents from the profile")
    sub.add_parser("check", help="hint when generated agents are stale")
    args = ap.parse_args(argv)
    root = Path(args.root).resolve() if args.root else find_root()
    if args.cmd == "stage":
        return cmd_stage(root, args.key)
    if args.cmd == "generate":
        return cmd_generate(root)
    return cmd_check(root)


if __name__ == "__main__":
    sys.exit(main())
```

def _templates_dir():
    return Path(__file__).resolve().parents[1] / "templates" / "agents"


def _render(template_text, model, effort, checksum):
    out = template_text
    if effort is None:
        out = re.sub(r"^effort: \{\{EFFORT\}\}\n", "", out, flags=re.M)
    else:
        out = out.replace("{{EFFORT}}", effort)
    return out.replace("{{MODEL}}", model).replace("{{CHECKSUM}}", checksum)


def cmd_generate(root):
    path = root / PROFILE_REL
    if not path.exists():
        print(f"models.py: no {PROFILE_REL} — nothing to generate", file=sys.stderr)
        return 1
    checksum = hashlib.sha256(path.read_bytes()).hexdigest()
    stages, warnings = parse_profile(path.read_text(encoding="utf-8"))
    for w in warnings:
        print(w, file=sys.stderr)
    agents_dir = root / ".claude" / "agents"
    dir_existed = agents_dir.is_dir()
    wrote_any = False
    for key, agent in AGENT_STAGES.items():
        rel = f".claude/agents/{agent}.md"
        row = stages.get(key)
        if row is None:
            print(f"model-profile: no valid '{key}' row — {agent}.md not regenerated", file=sys.stderr)
            continue
        if row["mechanism"] != "agent":
            print(
                f"model-profile: '{key}' row has mechanism '{row['mechanism']}' — {agent}.md not regenerated",
                file=sys.stderr,
            )
            continue
        target = agents_dir / f"{agent}.md"
        if target.exists() and not MARKER_RE.search(target.read_text(encoding="utf-8")):
            print(f"refused (user-owned, no marker): {rel}")
            continue
        template = (_templates_dir() / f"{agent}.md").read_text(encoding="utf-8")
        agents_dir.mkdir(parents=True, exist_ok=True)
        target.write_text(_render(template, row["model"], row["effort"], checksum), encoding="utf-8")
        print(f"wrote {rel}")
        wrote_any = True
    if wrote_any and not dir_existed:
        print("note: .claude/agents/ was just created — restart Claude Code before the new agents can be dispatched")
    return 0


def cmd_check(root):
    path = root / PROFILE_REL
    if not path.exists():
        return 0
    checksum = hashlib.sha256(path.read_bytes()).hexdigest()
    for agent in AGENT_STAGES.values():
        target = root / ".claude" / "agents" / f"{agent}.md"
        if not target.exists():
            continue
        m = MARKER_RE.search(target.read_text(encoding="utf-8"))
        if m and m.group(1) != checksum:
            print(MISMATCH_HINT)
            return 0
    return 0

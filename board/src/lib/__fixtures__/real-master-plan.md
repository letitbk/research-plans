# Immigration Attitudes (ISSP) — Master Plan

Last updated: 2026-07-02

## Project context

This project is a cross-national analysis of immigration attitudes using ISSP sample data. The core question is how support for immigration varies across countries, and what individual- and country-level factors are associated with that variation. Jane Doe is the sole researcher; there are no collaborators.

The data are already in the repository at `data/issp_sample.csv`. The current file is a small test sample with four columns (`id`, `country`, `year`, `support`); the working assumption is that the full extract will follow the same structure. The data are public survey data with no sensitivity constraints. An `analyze.R` placeholder script exists but contains no analysis yet.

The hard constraint is the deadline: end of July 2026. With a solo researcher and roughly four weeks, the work is scoped to three sequential components — cleaning, descriptives, and regression modeling — each planned and verified independently.
